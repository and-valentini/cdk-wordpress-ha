## To Deploy Code:
1. Checkout this git code
2. Install and Configure AWS CLI ( note that cdk deploy to your default profile)
3. Install Dependencies:
```
npm install
```
4. Check CDK Synth:
```
cdk synth
```
6. Deploy:
```
cdk deploy
```

# To clean up:
1. Run destroy command
```
cdk destroy
```
